There are many SKIRT fits being computed in here. This file aims at giving guidance on what each of these are. 


* **SKIRTOR_fit**: Main fitting for the general familiy of Hot DOGs. *Add p_chip_interpolation between models*

* **SKIRTOR_fit_onesided**: Fits used in the published version of the 2025 paper focusing on W0204-0506. Despite the name, the fitting is done for three cases: a single forward cone, a single receding cone and both cones simultaneously (see cone_types parameter). *Uses p_chip interpolation between models*
